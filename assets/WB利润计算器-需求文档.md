# WB（Wildberries）白关利润计算器 — 产品需求文档

> 平台：Coze 网站应用  
> 目标用户：WB 跨境电商运营人员、财务人员  
> 文档版本：v1.2  
> 日期：2026-04-04

---

## 一、产品背景与目标

俄罗斯电商平台 Wildberries（WB）采用白关（正规报关）模式进行销售，涉及多层费率（关税、增值税、平台佣金、物流、税费等）。现有团队通过 Excel 手工录入计算，效率低且容易出错。

**目标：** 构建一个在线利润计算 Web 应用，让运营人员输入产品关键参数，实时计算并展示含/不含广告的毛利润与毛利率，支持多 SKU 批量管理，辅助定价决策。

## 二、核心功能模块

### 模块 1：参数输入面板（Input Panel）

用户逐行（或批量）录入以下字段，系统自动完成所有计算。

#### 1.1 基础信息

| 字段名 | 说明 | 数据类型 | 备注 |
|--------|------|----------|------|
| 账号 | WB 账号名称 | 文本 | 例：WB |
| 产品链接 | WB 商品详情页 URL | 文本/链接 | 可选，点击可跳转 |
| SKU | 商品 SKU 编号 | 文本 | WB 平台编号 |
| 产品名称 | 商品名称 | 文本 | |
| WB 首图链接 | 商品主图 URL | 文本 | 可选，预览图 |
| 产品性质 | 新品推广 / 老品清货 / 维护 | 下拉选择 | |
| 类目 | WB 平台商品类目（俄文） | 下拉/搜索 | 关联佣金率自动填充 |

#### 1.2 价格与汇率

| 字段名 | 说明 | 数据类型 | 备注 |
|--------|------|----------|------|
| 实时售价（卢布） | WB 平台售价，卢布 | 数字 | |
| 卢布汇率 | 1 卢布 = ? 人民币，**自动从 API 获取实时汇率** | 数字 | 可手动覆盖单条 SKU |
| 平台售价（RMB） | 自动计算 = 售价(卢布) × 汇率 | 自动计算 | 只读 |

#### 1.3 成本项

| 字段名 | 说明 | 数据类型 | 备注 |
|--------|------|----------|------|
| 产品采购成本（RMB） | 实际采购价（含税 13%） | 数字 | |
| 海关申报成本（RMB） | 申报成本（通常为采购成本 +30%） | 数字 | 影响税基 |
| 头程物流 + 海外仓送仓费（RMB） | 含头程运费及送仓费 | 数字 | |
| 关税（RMB） | 由申报成本 × 关税税率计算 | 数字 | 白关模式下据实填写 |
| 进口增值税 22%（RMB） | 白关进口增值税 = (申报成本 + 关税) × 22% | 数字/自动 | 可手动覆盖 |
| 其他清关费 2%（RMB） | 其他清关杂费 = 申报成本 × 2% | 数字/自动 | |

#### 1.4 平台费用

| 字段名 | 说明 | 数据类型 | 备注 |
|--------|------|----------|------|
| WB 佣金率 | 按类目自动带入，支持手动覆盖 | 百分比 | 关联「类目佣金率」数据表 |
| WB 佣金（RMB） | = 平台售价(RMB) × 佣金率 | 自动计算 | 只读 |
| FBW 配送费 + 仓储费 + 验收费（RMB） | WB 仓储物流费合计 | 数字 | 手动输入 |
| 买方银行收单关税 1.5%（RMB） | = 平台售价(RMB) × 1.5% | 自动计算 | |
| 站内广告费（RMB） | 广告投放费用 | 数字 | 填 0 = 不含广告场景 |
| 退款率 | 预估退款比例 | 百分比 | 例：2% |

#### 1.5 平台回款与税费

| 字段名 | 说明 | 数据类型 | 备注 |
|--------|------|----------|------|
| 平台放款（RMB） | 扣除佣金、FBW费、买方收单关税后的回款 | 自动计算 | |
| 放款手续费 1%（RMB） | = 平台放款 × 1% | 自动计算 | |
| 15% 税费（RMB） | = (平台放款 − 申报成本 − 进口增值税) × 15% | 自动计算 | 俄罗斯企业所得税 |
| 5% 增值税（RMB） | = 平台售价(RMB) × 5% | 自动计算 | |
| 2.5% 回款手续费（RMB） | = 平台放款 × 2.5% | 自动计算 | |
| 国内回款金额（RMB） | = 平台放款 − 放款手续费 − 15%税费 − 5%增值税 − 2.5%回款手续费 | 自动计算 | 只读 |

### 模块 2：利润计算结果展示

每行 SKU 实时显示以下计算结果：

| 指标 | 计算逻辑 | 显示颜色 |
|------|----------|----------|
| **不含广告毛利润（RMB）** | 国内回款 − 采购成本 − 头程物流 − 关税 − 进口增值税 − 其他清关费 | 正数绿色，负数红色 |
| **不含广告毛利率** | 不含广告毛利润 ÷ 平台售价(RMB) | 同上 |
| **含广告毛利润（RMB）** | 不含广告毛利润 − 广告费 | 正数绿色，负数红色 |
| **含广告毛利率** | 含广告毛利润 ÷ 平台售价(RMB) | 同上 |
| **平台回款（RMB）** | 扣佣后回款 | 中性 |

> **计算公式详解（以白关运营利润表为准）：**
>
> ```
> 平台售价(RMB) = 售价(卢布) × 汇率
> 买方收单关税 = 平台售价(RMB) × 1.5%
> WB佣金 = 平台售价(RMB) × 佣金率
> FBW物流费 = 手动输入
> 平台放款 = 平台售价(RMB) − WB佣金 − FBW物流费 − 买方收单关税
> 放款手续费 = 平台放款 × 1%
> 15%税费 = (平台放款 − 申报成本 − 进口增值税) × 15%
> 5%增值税 = 平台售价(RMB) × 5%
> 回款手续费 = 平台放款 × 2.5%
> 国内回款 = 平台放款 − 放款手续费(1%) − 15%税费 − 5%增值税 − 2.5%回款手续费
>
> 不含广告毛利润 = 国内回款 − 采购成本 − 头程物流 − 关税 − 进口增值税 − 其他清关费
> 不含广告毛利率 = 不含广告毛利润 ÷ 平台售价(RMB)
> 含广告毛利润 = 不含广告毛利润 − 广告费
> 含广告毛利率 = 含广告毛利润 ÷ 平台售价(RMB)
> ```

### 模块 3：类目佣金率数据库

系统内置 WB 平台类目佣金率对照表（来源：表格「类目佣金率」sheet），支持：

- **按俄文类目名（Предмет）或中文名搜索**
- 用户在输入面板选择类目后，**佣金率自动填充**
- 支持管理员手动更新/维护佣金率数据

佣金率范围：0.5% ～ 27.5%（依类目不同）

示例数据：

|---------|---------|-------|
| Фены | 吹风机 | 14.5% |
| Комплекты клавиатур и мышей | 键鼠套装 | 20.5% |
| Микрофоны | 麦克风 | 7.5% |

### 模块 4：多 SKU 管理（表格视图）

- 支持**同时管理多条 SKU 记录**，以表格行形式展示

## 三、页面结构与交互设计

### 页面布局

```
┌─────────────────────────────────────────────────┐
│  顶部导航栏：WB 利润计算器 | 设置 | 导出        │
├─────────────────────────────────────────────────┤
│  全局参数快捷栏：汇率 | 广告费率（可快速修改）   │
├─────────────────────────────────────────────────┤
│  SKU 列表（表格）                               │
│  ┌──┬────────┬──────┬───────┬──────┬──────────┐ │
│  │# │产品名  │售价  │不含广 │含广  │操作      │ │
│  │  │        │卢布  │毛利率 │毛利率│编辑/删除 │ │
│  └──┴────────┴──────┴───────┴──────┴──────────┘ │
│  [+ 新增 SKU]                                   │
├─────────────────────────────────────────────────┤
│  SKU 详情展开面板（点击行展开/收起）            │
│  - 完整输入表单（分组：基础信息/成本/平台费用） │
│  - 计算结果明细（逐项显示各费用和利润）         │
└─────────────────────────────────────────────────┘
```

### 交互要求

1. **实时计算**：任意输入字段变更后，结果立即更新（无需点击按钮）
2. **类目联动**：选择类目后，佣金率字段自动填充，用户可手动覆盖
3. **利润警示**：毛利率低于 10% 时，数值显示为红色并加警示图标
4. **字段说明**：鼠标悬浮在字段标签上，显示计算公式和说明（Tooltip）
5. **含广/不含广切换**：在汇总表格可一键切换展示维度
6. **实时汇率自动更新**：页面加载时自动调用汇率 API 获取最新 RUB/CNY 汇率，全局刷新所有 SKU；顶部显示汇率更新时间，支持手动刷新按钮；单条 SKU 可独立锁定汇率

## 五、汇率 API 集成规范

### 推荐 API

优先使用以下免费/低成本汇率接口（RUB → CNY）：

| API | 地址 | 免费额度 | 备注 |
|-----|------|---------|------|
| ExchangeRate-API | `https://api.exchangerate-api.com/v4/latest/RUB` | 1500次/月 | 无需 Key，直接调用 |
| Frankfurter | `https://api.frankfurter.app/latest?from=RUB&to=CNY` | 无限制 | 欧洲央行数据，免费开源 |
| Open Exchange Rates | `https://openexchangerates.org/api/latest.json` | 1000次/月 | 需申请免费 Key |

**推荐优先使用 Frankfurter**（无需注册，无频率限制，数据可靠）。

### 调用逻辑

```
请求：GET https://api.frankfurter.app/latest?from=RUB&to=CNY
响应示例：
{
  "base": "RUB",
  "date": "2026-04-04",
  "rates": { "CNY": 0.0842 }
}

汇率取值：rates.CNY
```

### 更新策略

| 场景 | 行为 |
|------|------|
| 页面首次加载 | 自动调用 API，获取最新汇率 |
| 顶部「刷新汇率」按钮点击 | 重新调用 API，全局更新 |
| API 调用失败 | 使用上次缓存汇率，并显示警告提示（"汇率更新失败，使用上次数据：0.085，更新于 XX:XX"） |
| 单条 SKU 手动锁定汇率 | 该 SKU 使用本地输入值，不受全局汇率更新影响（字段旁显示🔒图标） |
| 汇率显示位置 | 顶部栏固定展示：「当前汇率：1 RUB = 0.0842 CNY｜更新于 14:32 ↻」 |

## 六、计算模式说明

本应用**仅实现白关模式**（正规报关），对应最新 Sheet「财务最新WB 2026-3-27」，包含完整税制：关税 + 进口增值税 22% + 俄罗斯企业所得税 15% + 5% 增值税。

灰关模式**不在本期范围内**。

| 类型 | 要求 |
|------|------|

| 多用户 | 不在本期范围内 |
